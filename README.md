# Display Panel v5.0.11

Personal information dashboard for multi-device displays.

## Features
- Real-time weather (3 locations)
- Google Calendar integration
- Stock market tracking
- Currency alerts
- News feeds (Chinese + International + Traffic)
- Live news stream (Now News 332)
- Bible verse rotation

## Setup
1. Open `index.html` in browser
2. Click ⚙️ Settings
3. Add API keys:
   - Weather: OpenWeatherMap
   - Stocks: TwelveData + Finnhub
   - Calendar: Google Apps Script URL
4. Save & enjoy!

## Supported Devices
- Large TVs (1920×1080+)
- Tablets (2000×1200)
- Foldable phones (2640×1080)
